# Contributing to the HCI Chatbot-Human Interaction Project

First off, thank you for taking the time to contribute! Whether you are an HCI researcher, data scientist, or open-source enthusiast, your insights help improve this research.

## How Can I Contribute?

### 1. Expanding the Literature Matrix
If you find relevant academic papers that investigate human-chatbot interaction dynamics, feel free to suggest adding them:
* Open an issue discussing the paper.
* Extract the variables following our current matrix format.
* Append them to the `Literature Matrix.xlsx` and ensure proper binary coding rules are maintained.

### 2. Improving the Machine Learning Pipeline
If you want to test more advanced predictive models (e.g., Deep Learning, alternative ensemble methods) or optimize hyperparameters:
* Fork the repository.
* Create a new branch describing your model (e.g., `feature/add-xgboost`).
* Update the machine learning notebook and ensure you document your final accuracy scores.

### 3. Reporting Bugs or Discrepancies
If you notice mathematical errors, preprocessing logical flaws, or broken code in the `.ipynb` files:
* Open an **Issue** explaining the discrepancy.
* Provide reproducible examples or reference the exact lines of code.

## Pull Request Process

1. Fork the repo and create your branch from `main`.
2. Ensure your code is well-commented and aligns with the existing project structure.
3. If you modified the dataset, update the corresponding documentation or preprocessed sheets.
4. Submit a Pull Request with a clear description of your changes and why they improve the project.

## Code of Conduct
Please be respectful and collaborative in all issues, pull requests, and discussions. We are all here to learn and advance HCI research!
